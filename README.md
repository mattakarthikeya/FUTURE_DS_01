# Business Sales Performance Dashboard

This project analyzes business sales data to identify:

- Revenue trends
- Top-selling products
- High-value categories
- Regional performance

## Tools Used
Power BI

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales Trend Analysis
- Top 10 Selling Products
- Sales by Category
- Sales by Region

## Key Insights
- Technology category generates the highest sales.
- West region contributes the most revenue.
- Canon imageCLASS is one of the top-selling products.
